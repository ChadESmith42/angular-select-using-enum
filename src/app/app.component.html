Using Angular's KeyValue Pipe

Pick a State:

The value is {{stateSelect.value}}.

The displayed value is {{object[stateSelect.value]}}.

We have an enum of US State names formatted as

IN = "Indiana"

Using the keyvalue pipe that is build into Angular, we can use the enum in a select HTML element.

First, I created an object to hold the state enum values and assign the enum to that object:

object: { [ key: string ]: string } = State;

Then, I loop through each value in the enum and add the keyvalue pipe. Since each value of the enum is now broken into an object with a key and value property, I can assign each of those values to the control.

<select [formControl]="stateSelect"> <option *ngFor="let state of object | keyvalue" [value]="state.key">{{state.value}}</option>
</select>
